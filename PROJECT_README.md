# Walid Elmansouri Portfolio

A modern, minimalist portfolio website built with React and Tailwind CSS featuring a clean white design with black accents.

## Features

- ✨ Smooth animations (floating profile photo)
- 📱 Fully responsive design
- 🎨 Clean white background with black text and borders
- ⚡ Built with React + Vite + Tailwind CSS
- 🎯 Centered layout with professional typography

## Sections

- **Hero Section** - Name, title, and animated profile photo
- **About Me** - Professional summary
- **Skills** - Design tools and languages
- **Projects & Experience** - Recent work and workshops
- **Education** - Academic background
- **Contact** - Location, email, and phone

## Installation

1. Navigate to the project directory:

```bash
cd "c:\Users\ULTRA PC\Desktop\projects\walid port"
```

2. Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

Create an optimized production build:

```bash
npm run build
```

## Preview

Preview the production build locally:

```bash
npm run preview
```
